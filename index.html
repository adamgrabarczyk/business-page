Oferta

Strony internetowe

Wykonuję strony responsywne, elastyczne i dopasowane oraz dedykowane wersje mobilne. Projektuję strony Internetowe, tworzye własne systemy zarządzania stroną internetową.

Aplikacje WWW

Wykonuję aplikacje internetowych do zarządzania stronami i portalami www. Stawiam w nich przede wszystkim na funkcjonalność i intuicyjność obsługi.

Aplikacje mobline

Tworze aplikacje mobilne w technologii React Native. Dzięki zastosowaniu technologii webowych, jedna wersja aplikacji dostosowana jest do dwóch najpopularnieszych systemów operacyjnych takich jak iOS i Android.

Rozwiązania dedykowane

Dedykowana aplikacja internetowa to system informatyczny, wykorzystujący platformę internetową. Powstaje z myślą o konkretnym zastosowaniu i rozwiązaniu określonych problemów.

Reklama

Projektuję banery internetowe,one-page'owe wizytówki i filmiki promujące na Youtube.

Obsługa produktu

Zajmuję się aktualizacją wykonanych przezemnie produktów, jak i równierz rozwiązań zewnętrznych. Oferuje administowanie Państwa stronami, zarządzanie hostingami i domeną. Jeżeli zajdzie taka potrzeba, służę również doradztwem na szerokiej płaszczyźnie rozwiązań IT.

Technologie z których korzystam najczęściej

HTML5

Język html stosuje się do tworzenia stron internetowych. Przeglądarki internetowe odczytują składnię html i interpretują ją, wyświetlając stronę w przeglądarce.

CSS

Za pomocą CSS'a określamy wygląd różnych elementów HTML znajdujących się na danej stronie internetowej.

JavaScript

Skryptowy język programowania, który umożliwia zaawansowaną interakcję z użytkownikiem, oraz strowanie logiką aplkiacji.

Bootstrap

framework CSS, rozwijany przez programistów Twittera,ułatwia tworzenie interfejsu graficznego stron oraz aplikacji internetowych.

React JS

React to biblioteka od developerów takich portali jak Facebook czy Instagram, która służy do tworzenia komponowalnych interfejsów użytkownika.

Node JS

Jest frameworkiem open source. Środowisko uruchomieniowe zaprojektowane do tworzenia wysoce skalowalnych aplikacji internetowych.

PHP

PHP jest najczęściej stosowany do tworzenia skryptów po stronie serwera WWW.PHP zastosować można również do przetwarzania danych.Jest językiem open source.

Java JEE

Aplikacje webowe w języku Java można tworzyć dzięki specyfikacji JEE . Jest to zbiór mechanizmów wykorzystywanych przy budowaniu aplikacji webowych.
